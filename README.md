# Chloe Birthday Website

A modern romantic birthday website for Chloe, built with React, TypeScript, Vite, and Tailwind CSS.

## Run Locally

```bash
npm install
npm run dev
```

## QR Code

Set `VITE_SITE_URL` in a `.env` file to the final deployed URL. The site includes a printable QR card near the bottom of the page.

```bash
VITE_SITE_URL=https://your-real-url.com
```

For a quick deployment, Vercel, Netlify, or GitHub Pages will all work well with the production build:

```bash
npm run build
```
